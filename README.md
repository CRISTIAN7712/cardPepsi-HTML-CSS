# Interactive Product Card - Pepsi

## Descripción del Proyecto

Este proyecto es una tarjeta de producto interactiva para **Pepsi** que muestra una animación al pasar el cursor sobre ella. La tarjeta incluye:

- Un círculo con el logo de Pepsi que se transforma en un fondo azul  
- Una lata de Pepsi que aparece con una animación de rotación  
- Información del producto que se revela  
- Un botón de **"Shop Now"**

---

## Características Principales

- **Efectos de hover interactivos**: La tarjeta se expande y revela más contenido al pasar el cursor  
- **Animaciones fluidas**: Transiciones CSS para todos los elementos  
- **Diseño responsivo**: Se adapta a diferentes tamaños de pantalla  
- **Efectos visuales**: Sombras y transformaciones 3D  

---

## Tecnologías Utilizadas

- **HTML5**  
- **CSS3** (con transformaciones, transiciones y filtros)  
- **Google Fonts** (Poppins)

---

## Instalación

No se requiere instalación especial. Solo abre el archivo `index.html` en tu navegador.

---

## Cómo Usar

1. Abre el archivo HTML en tu navegador  
2. Pasa el cursor sobre la tarjeta para ver la animación  
3. Haz clic en **"Shop Now"** (enlace de ejemplo)

---

## Personalización

Puedes modificar los siguientes aspectos fácilmente:

- **Color principal**: Cambia la variable CSS `--clr` en el selector `body`  
- **Imágenes**: Reemplaza `image.png` (logo) y `image1.png` (lata del producto)  
- **Texto**: Modifica el contenido en la sección `.content`  
- **Tamaños**: Ajusta las dimensiones en las clases `.card`, `.circle`, etc.
